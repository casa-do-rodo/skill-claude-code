Opus 4.7

ui-subagent

Subagente especializado em UI com modelo Opus 4.7 fixo. Garante qualidade visual máxima em todas as tasks de interface, independente do modelo da sessão principal.

  • Modelo Opus 4.7 — sempre, sem downgrade
  • Skills visuais pré-carregadas (frontend-design, svg-animations, design-taste-frontend)
  • Tools restritas a Read/Edit/Write — foco total em código visual
  • Invocado automaticamente para tasks de UI no subagent-driven-dev
Gate obrigatório

frontend-audit-gate

Gate de qualidade obrigatório antes do ship. Orquestra 3 audits em paralelo, agrega findings por prioridade e aplica correções via 6 subagentes Opus paralelos.

  • 3 audits simultâneos: redesign, web-guidelines, svg-animations
  • Findings classificados: Critical → High → Medium → Polish
  • Aplicação paralela: 6 subagentes Opus em ~67 segundos
  • Obrigatório antes de finishing-a-development-branch
